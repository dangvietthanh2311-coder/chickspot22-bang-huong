# ChickSpot 22 — Bảng hướng quyết định

Trang tĩnh (static site) hiển thị bảng hướng xây dựng thương hiệu ChickSpot 22.

- `index.html` — toàn bộ nội dung (HTML + CSS + JS inline, self-contained).
- `vercel.json` — cấu hình Vercel tối thiểu cho static hosting.

## Deploy
Trang này là static thuần, không cần build. Vercel tự nhận diện và phục vụ `index.html` ở trang gốc.
